Java基本知识

注意事项

  • 默认double类型，long类型和float声明是需要加上大写的L或F

  • 大类型可以转为小类型，小类型转换要强制转换,注意一出

  int num = (int)100L

  • && 和 || 具有短路效果，左边可以判断最终结果，停止执行

  • 方法定义

    public static void cook(){
   //方法体
}

方法重载

  • 方法重载与下列因素有关
    • 参数个数不同
    • 参数类型不同
    • 参数的多类型顺序不同
  • 方法重载与下列因素有关
    • 与参数的名称无关
    • 与方法的返回值类型无关
public class OverLoadStudy {

    public static void main(String[] args) {
        System.out.println(sum(2.2, 3));
        System.out.println(sum(10,10));
        System.out.println(sum(10,10, 10));
    }

    public static int sum(int a, int b){
        return a+b;
    }
    //    参数类型不同
    public static int sum(double a, int b){
        return (int)(a+b);
    }
    //    参数类型不同
    public static int sum(int a, int b, int c){
        return (int)(a+b);
    }
}

创建数组

public class ArrayStudy {
    public static void main(String[] args) {
        // 动态创建
        int[] array = new int[10];
        // 静态创建
        int[] array1 = new int[]{1,2,3};
        // 静态省略创建
        int[] array2 = {1,2,3,4};
        // 静态分步创建
        int[] array3;
        array3 = new int[]{3,43,2};
        // 动态分步创建
        int[] array4;
        array4 = new int[10];
        array4[1] = 10;
        System.out.println(array1[1]);
        System.out.println(array2[1]);
        System.out.println(array3[1]);
        System.out.println(array4[1]);
    }

}

Java的内存划分

  • 栈（Stack):
    • 存放的都是方法中的局部变量。方法运行一定要在栈当中运行。
    • 局部变量：方法的参数，或者方法{}内部的变量
    • 作用域：一旦超出作用域，立刻从从栈内存当中消失。
  • 堆(Heap):
    • 凡事new出来的东西都在堆当中。
    • 堆内存里面的东西都有一个地址值：16进制
    • 堆内存里面的数据，都有默认值，规则：
      • 正数 默认0
      • 浮点数 默认0.0
      • 字符 默认’\u0000’
      • 布尔 默认false
      • 引用类型 默认null， 如数组
  • 方法区(Method Area): 存储.class相关信息，包含方法的信息。
  • 本地方法栈(Native Method Stack):与操作系统相关。
  • 寄存器（pc Register): 与CPU相关。
